# Cisco Packet Tracer Labs

This repository contains selected network simulation labs I completed using Cisco Packet Tracer.

## 📁 Projects Included:

### 1. Basic Network Connection
This project demonstrates a simple network setup connecting end devices to a remote server.

- One device is connected wirelessly, and another is connected via cable  
- Both devices connect to a wireless router  
- The router connects to a cable modem, then to the internet (cloud)  
- The network communicates with a remote server  

This lab helped me understand how devices connect through different network components to access services over the internet.

---

### 2. Smart Home IoT Network
This project simulates a smart home environment where multiple IoT devices are connected through a home gateway.

- Devices are connected using both wired and wireless connections  
- A home gateway is used to manage communication between devices  
- A tablet is used to control smart devices (turning them on/off)  
- Demonstrates how IoT systems can process data locally (edge computing) instead of relying on the cloud  

This setup highlights how local processing improves response time and reduces dependency on internet connectivity.
